# 3D Work-Shop

3D Work-Shop is a web application where users can register, log in, upload FBX files, browse uploaded 3D objects, view object previews, download files, and add comments.

The project uses:
- Node.js with Express
- SQLite databases for users and objects/comments
- Plain HTML, CSS, and JavaScript
- Three.js for FBX previews

Main features:
- User and admin authentication
- Persistent storage in two databases
- Object upload and listing
- Static 3D preview cards and rotating detail preview
- Comment section per object
- Admin page for managing users and objects
- Light and dark theme toggle

Run locally:
1. Install dependencies with `npm install`
2. Start the server with `npm start`
3. Open `http://localhost:3000`

Default accounts:
- user / password
- admin / password
